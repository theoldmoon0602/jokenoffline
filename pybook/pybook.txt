# プログラミングとは

　プログラミングとはプログラムをつくることです。

# コメント

　プログラム中にはコメントを書くことができます。コメントとは、Pythonが無視をする文字列のことです。コメントとしてなにを書こうとも、それはPythonプログラムの実行結果に影響を与えません。コメントはソースコードの意味を記述するために使われます。

　ソースコード中の、#から行末までがコメントと認識されます。

```
print("Hello, World!") # Hello, World!と表示

```

　また、"""や'''で囲まれた領域もコメントと認識されます。

```
"""
This is a comment region.
"""
```

# print関数

　関数が何か、printが何かという話は後回しにして、これがないとつらいので、print関数を紹介します。print関数は、引数として受け取った値を標準出力に書き出す関数です。

　次のように使います。

```
print("some string object")
print(1)
```

 　実行結果はこうなります。

```
some string object
1
```

# 変数

```
x = 10
print(x)
x = 11
print(x)
x = 12
print(x)
x = "string"
print(x)
```

　変数は、書き換え可能な値です。状態を保存するときによく使います。たとえば、ゲームにでてくるHPやSCOREの状態の記憶には変数を使っているはずです。変数を使ってある状態を保存しておいて、更新したり、利用したりします。

　変数に値を入れるには「代入」という操作を行います。変数から値を取り出すのに特別な操作は必要ありません。いままでのリテラルと同じように使うことができます。


# 代入

　代入は、変数に値を入れる操作のことです。```x = 10```として、変数xに10という値を設定することを、「xに10を代入する」といいます。

# 演算

```
print(1 + 1)
print(1 * 2)
print(1 - 3)
print(9 / 5)
print(9 // 5)
print(9 % 5)
print(5 ** 2)
print(1 + 2 * 3)
print((1 + 2) * 3)
```

　演算は計算のことです。演算をするためには演算子を用います。演算子とは、+や-の記号のことです。Pythonでは、数学と同じように計算が出来ます。加減乗除は+-*/、計算の優先順位は()で指定できます。

　//は切り捨てる除算、%は剰余算、**は累乗の計算です。
